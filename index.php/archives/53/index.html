Git常用命令

April 21, 2019

查看git配置信息

git -l

配置当前仓库下提交者的用户名和邮箱

git config user.name "提交者的名称"
git config user.email "邮箱"

配置全局提交者的用户名和邮箱

git config --global user.name "用户名"
git config --global user.email "邮箱"

克隆远程仓库

git clone <地址>

查看工作区状态

git status

添加文件到暂存区

git add <文件名>

提交文件到本地分支

git commit -m <"注释"> <文件名称>

将本地分支推送到远程分支

git push

将远程仓库的代码拉取到本地仓库

git pull

查看提交日志

git log 或 git log --pretty=oneline

回退代码

git reset --hard <记录id，可通过日志查看>

撤销回退

git reflog
git reset --hard <记录id，通过上述命令查看>

撤销修改

git checkout -- <文件名>

撤销添加文件到暂存区

git reset HEAD -- <文件名>

查看当前分支

git branch

创建分支

git branch <分支名称>

切换分支

git checkout <分支名称>

创建且切换分支

git checkout -b <分支名称>

将分支下的内容提交到远程仓库

git push origin <分支名称>

拉取远程分支到本地分支

git pull origin <分支名称>

将指定分支与当前分支合并

git merge <分支名称>
